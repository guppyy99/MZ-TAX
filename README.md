# MZ-TAX 하드카피 파일명 자동 변경 프로그램

회계 업무 처리 시 광고 관련 서류 파일명을 제출 규칙에 맞게 자동으로 변경해주는 웹 애플리케이션입니다.

## 주요 기능

- 🎯 다양한 회계 서류 파일명 자동 인식 (지출결의서, 광고 정산 품의서, 세금계산서, 견적서 등)
- 🔢 고정값 순서에 맞는 자동 넘버링 (0, 1, 2-1, 2-2, 3-1, 3-2, 4, 5)
- 📝 파일명 규칙에 맞는 자동 변경
- 🔍 파일명에서 정보 자동 추출 (광고명, 월, 을 업체명 등)
- 🤖 **AI PDF 분석** (OpenAI GPT-5-mini) - PDF 내용을 읽고 자동으로 정보 추출
- 📂 드래그 앤 드롭 지원
- 🎨 라이트톤 UI (OpenAI 대시보드 스타일)
- 🔤 Pretendard 폰트 사용

## 시작하기

### 초기 설정 (선택사항 - AI 기능 사용 시)

1. **OpenAI API 키 발급**
   - [OpenAI API Keys 페이지](https://platform.openai.com/api-keys)에서 API 키 발급
   - GPT-5-mini 모델 사용 (한국어 최적화, 매우 저렴)

2. **API 키 설정**
   - 우측 상단 ⚙️ 설정 버튼 클릭
   - API 키 입력 (브라우저에만 저장됨, 안전)
   - "연결 테스트" 버튼으로 확인
   - "저장" 버튼 클릭

### 사용 방법

#### 방법 1: AI 자동 분석 (권장)

1. **웹 브라우저에서 `index.html` 파일 열기**
   - 더블클릭하거나 브라우저로 드래그하여 열기

2. **PDF 파일 선택**
   - **파일 선택 버튼**: 클릭하여 PDF 파일 선택
   - **드래그 앤 드롭**: PDF 파일을 프로그램 창으로 끌어다 놓기

3. **AI로 자동 분석** 🤖
   - "AI로 자동 분석" 버튼 클릭
   - PDF 내용을 읽고 자동으로 광고명, 월, 업체명 등을 추출
   - 입력 필드에 자동으로 채워짐

4. **미리보기 확인 및 수정**
   - 변경될 파일명을 미리보기에서 확인
   - 필요시 수동으로 수정 가능

5. **처리 실행**
   - "처리 실행" 버튼을 클릭하면 변경된 파일명으로 자동 다운로드

#### 방법 2: 수동 입력

1. **웹 브라우저에서 `index.html` 파일 열기**

2. **파일 선택** (모든 형식 지원)
   - PDF, JPG, PNG, XLSX 등 모든 파일 형식 지원

3. **정보 수동 입력**
   - **광고명**: 광고명 입력 (파일명에서 자동 추출 시도)
   - **월**: 월 입력 (파일명에서 자동 추출 시도)
   - **갑 업체**: 갑 업체명 (기본값: 메가존)
   - **을 업체**: 을 업체명 (파일명에서 자동 추출 시도)

4. **미리보기 확인**
   - 변경될 파일명을 미리보기에서 확인

5. **처리 실행**
   - "처리 실행" 버튼을 클릭하면 변경된 파일명으로 자동 다운로드

## 제출 순서 (고정값)

파일 유형별로 고정된 넘버링이 적용됩니다:

| 번호 | 파일 유형 | 설명 |
|------|-----------|------|
| **0** | 사업자등록증, 통장사본 등 | 광고/갑을 구분 불필요 |
| **1** | 지출결의서 | 지출 결의 문서 |
| **2-1** | 광고 정산 품의서 / 일반 범용 품의서 / 외주구매품의서 | 주요 품의서 |
| **2-2** | 선지급 품의서 | 선지급이 있는 경우 |
| **3-1** | 세금계산서 (을 발송) | 을 업체가 발신 |
| **3-2** | 세금계산서 (갑 발송) | 갑 업체가 발신 (대행 수수료 등) |
| **4** | 게재신청서 / 견적서 / 거래명세서 | 부가 서류 |
| **5** | 결과물 (보고서, 리포트, 게재 보고서 등) | 최종 결과물 |

## 파일명 규칙

### 기본 형식
```
파일유형_월_광고명_갑업체명_을업체명_날짜
```

### 상세 규칙

#### 0. 사업자등록증, 통장사본 등
```
0. 파일유형_업체명_날짜
```
**예시**: `0. 통장사본_프리비알_251125`

#### 1. 지출결의서
```
1. 지출결의서_8월_광고명_갑업체명_을업체명_날짜
```
**예시**: `1. 지출결의서_8월_비티진_메가존_프리비알_251125`

#### 2-1. 광고 정산 품의서
```
2-1. 광고정산품의서_8월_광고명_갑업체명_을업체명_날짜
```
**예시**: `2-1. 광고정산품의서_8월_비티진_메가존_프리비알_251125`

#### 2-2. 선지급 품의서
```
2-2. 선지급품의서_8월_광고명_갑업체명_을업체명_날짜
```
**예시**: `2-2. 선지급품의서_8월_비티진_메가존_프리비알_251125`

#### 3-1. (을 발송) 세금계산서
```
3-1. 세금계산서_[을업체명 발신]_8월_광고명_갑업체명_을업체명_날짜
```
**예시**: `3-1. 세금계산서_[프리비알 발신]_8월_비티진_메가존_프리비알_251125`

#### 3-2. (갑 발송) 세금계산서
```
3-2. 세금계산서_[갑업체명 발신]_8월_광고명_갑업체명_을업체명_날짜
```
**예시**: `3-2. 세금계산서_[메가존 발신]_8월_비티진_메가존_프리비알_251125`

#### 4. 견적서, 게재신청서, 거래명세서
```
4. 파일유형_8월_광고명_갑업체명_을업체명_날짜
```
**예시**:
- `4. 견적서_8월_비티진_메가존_프리비알_251125`
- `4. 거래명세서_8월_비티진_메가존_프리비알_251125`
- `4. 게재신청서_8월_비티진_메가존_프리비알_251125`

#### 5. 결과물 (리포트, 게재 보고서 등)
```
5. 결과물_8월_광고명_갑업체명_을업체명_날짜
```
**예시**: `5. 결과물_8월_비티진_메가존_프리비알_251125`

### 중요 사항
- 날짜는 시스템의 현재 날짜가 사용됩니다 (YYMMDD 형식)
- 월은 `08월` → `8월`로 표기 (0 제거)

## 주요 특징

### 🔍 자동 정보 추출
- 파일명에서 광고명, 월, 을 업체명을 자동으로 추출
- 첫 번째 파일의 정보가 입력 필드에 자동으로 채워짐
- 모든 띄어쓰기 패턴에 감응하여 추출

### 📋 띄어쓰기 감응 파싱
- 언더스코어(`_`), 공백, 하이픈(`-`), 점(`.`) 등 모든 구분자 지원
- 파일명에서 정보 추출 시 띄어쓰기와 무관하게 인식

### 🏢 을 업체명 자동 추출
- 파일명에서 메가존이 아닌 업체명을 자동으로 찾아서 을 업체명으로 설정
- 프리비알, 메타버즈, 비티진 등 다양한 업체명 인식

### 📝 파일명 생성 규칙
- 입력 필드의 값이 항상 우선 사용됨
- 파일명에서 추출한 정보는 입력 필드가 비어있을 때만 사용

## 🤖 AI 기능 상세

### PDF 자동 분석
- **사용 모델**: OpenAI GPT-5-mini
- **한국어 최적화**: 한국어 회계 서류 전문 분석
